# RailReporters — Synthèse progression V2 Supabase

## Objectif

Ce document résume l’état actuel de la préparation V2 Supabase de RailReporters.

La version publique V1.2 sur `railreporters.com` reste stable et inchangée.

---

## 1. Statut global

La V2 Supabase est en préparation avancée.

Le site local peut déjà :

- se connecter à Supabase Auth ;
- lire le profil utilisateur ;
- reconnaître le rôle admin ;
- créer un report Supabase ;
- créer plusieurs sections Supabase ;
- uploader une photo dans Supabase Storage ;
- créer une ligne dans `report_photos`;
- créer un commentaire Supabase ;
- relire un report complet avec auteur, sections, photo et commentaire.

Statut global : en très bonne progression.
